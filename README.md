## Installation

The latest precompiled binaries are available on [github](https://github.com/figsoda/patsh/releases/latest).

Alternatively you can install patsh from [crates.io][crate] with cargo.

```sh
cargo install patsh
```


## Building from source

```sh
cargo build --release
```
